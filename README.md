## 기획 의도

현대 기업 환경에서 **인적 자원의 효율적 관리**와  
근태·급여의 **자동화된 프로세스**는 기업 경쟁력의 핵심입니다.  

본 프로젝트는 직원 등록부터 출퇴근, 휴가, 급여, 증명서 발급까지  
**인사 관리 전 과정을 통합 지원**하는 ERP HRMS 구축을 목표로 합니다.
<br>


## 주요 기능

- 직원 등록 및 인사 정보 관리  
- QR 코드 기반 출퇴근 자동 처리  
- 외출·출장 등 다양한 근태 상황 관리  
- 휴가 신청 및 관리자 승인 프로세스  
- 한 달 근태 기록 기반 급여 자동 산출  
- 증명서(재직, 경력, 급여 명세) 발급 기능
